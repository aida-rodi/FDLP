Método de array .filter()

- El método .filter() es un método iterativo, por lo que va a llamar a una función callback (que se le pasa por paréntesis) una vez por cada elemento de un array, y va a comprobar que cumplan una condición.


- El método .filter() crea un nuevo array en el que va a ir metiendo los elementos del array inicial que cumplan dicha condición.


- Los elementos que no cumplan la condición no se van a incluir en el nuevo array creado por .filter().


- Si ningún elemento cumple la condición, .filter() retorna un array vacío.
